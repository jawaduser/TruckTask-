📦 TruckTask

Logistics & Transport Management Mobile App

A powerful logistics/transport management mobile application designed for delivery companies. Admin users can create trips, manage deliveries, and track operations efficiently.

✨ Features

  • ✅ Trip Management System
  • ✅ Delivery Tracking
  • ✅ Admin Dashboard
  • ✅ User Authentication & Authorization
  • ✅ Real-time Updates
  • ✅ SQLite Database

🛠️ Technology Stack

C# .NET MAUI SQLite MVVM
