# TruckTask-
a logistics/transport management mobile application using C# + .NET MAUI + SQLite following an MVVM structure.this a management system app for delivery company, where admin users can create a trip and assign that trip to a driver plus the truck that will take the trip, trips its the assignment or the order that the will be delivered with location and date and another info, drivers can see their trips assigned to them marking it as completed or pending, and can sending requests to the admin to ask for additional info or for asking for maintenance of the trucks, and admin can the availability of such of drivers or trucks. you can download the files and opening it in Visual studio and run it.
