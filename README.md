# TruckTask- 🚚

a logistics/transport management mobile application using C# + .NET MAUI + SQLite following an MVVM structure. This is a management system app for delivery company, where admin users can create trips and manage deliveries efficiently.

## 🛠️ Technology Stack

- **Language:** C#
- **Framework:** .NET MAUI
- **Database:** SQLite
- **Architecture:** MVVM (Model-View-ViewModel)
- **Platform:** Cross-platform Mobile App

## 📋 Getting Started

### Prerequisites
- Visual Studio 2022 or later
- .NET 8.0 or higher
- .NET MAUI workload

### Installation

1. Clone the repository:
```bash
git clone https://github.com/jawaduser/TruckTask-.git
cd TruckTask-
```

2. Restore dependencies:
```bash
dotnet restore
```

3. Build the project:
```bash
dotnet build
```

4. Run the application:
```bash
dotnet maui run
```
